# English OSS V1 Known Limitations

This release focuses on first impression, onboarding, and the happy path.

Deliberately not fully addressed in v1:

- every backend error message is not translated
- every internal system log is not translated
- some edge-case states may still expose mixed-language text
- the app is not yet a complete internationalization pass

What v1 does optimize:

- clear GitHub positioning
- English-first onboarding on the landing flow
- happy-path workflow copy
- optional Codex and Claude Code helper guidance

This is intentional scope control, not an oversight.
