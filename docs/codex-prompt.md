# Codex Prompt

Use this prompt when you want Codex to improve your simulation brief before you run MiroFish.

```text
I am preparing a MiroFish simulation brief.

Scenario type:
[pricing change / product launch / policy or org change / reputation or crisis / custom]

Draft brief:
[paste your current brief here]

Source material available:
- [list the documents you have]

Please help me do three things:
1. Rewrite the brief so it is concrete, stakeholder-aware, and ready for a multi-agent simulation.
2. Point out the most important missing context or missing documents.
3. Return a final brief in this format:

Scenario:
[text]

Timing:
[text]

Key stakeholders:
[text]

Main question:
[text]

Success or risk to evaluate:
[text]
```

Tips:

- Be specific about what decision, launch, or policy is changing.
- Name the stakeholder groups you care about.
- Ask for a brief you can paste directly into MiroFish.
